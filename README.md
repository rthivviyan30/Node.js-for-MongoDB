# 🚀 Node.js for MongoDB

A beginner-friendly Node.js project demonstrating how to connect and interact with a MongoDB database. This repository covers the fundamentals of using MongoDB with Node.js, making it an excellent starting point for developers learning backend development and NoSQL databases.

---

## 📖 Overview

This project showcases the essential operations required to work with MongoDB in a Node.js environment. It demonstrates how to establish a database connection, perform CRUD (Create, Read, Update, Delete) operations, and organize backend code efficiently.

Whether you're new to Node.js or exploring MongoDB, this project provides a simple and practical foundation.

---

## ✨ Features

* 🔗 Connect Node.js to MongoDB
* ➕ Create new documents
* 📄 Retrieve documents from the database
* ✏️ Update existing documents
* ❌ Delete documents
* 📂 Clean and organized project structure
* ⚡ Easy to understand for beginners

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/rthivviyan30/Node.js-for-MongoDB.git
```

### 2. Navigate to the project folder

```bash
cd Node.js-for-MongoDB
```

### 3. Install dependencies

```bash
npm install
```

### 4. Configure MongoDB

If you're using a MongoDB connection string, create a `.env` file and add:

```env
MONGODB_URI=your_mongodb_connection_string
```

---

## ▶️ Run the Project

```bash
node index.js
```

Or, if using Nodemon:

```bash
nodemon index.js
```
